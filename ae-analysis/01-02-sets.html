Exercise 3 (Proof of Proposition 2.7)

Proposition 2.7 Let {Aα ; α∈A} and {Bβ ; β∈B} be families of subsets of a set X.

(i) Associativity:

  • (⋂αAα)∩(⋂βBβ)=⋂(α,β)Aα∩Bβ

  • (⋃αAα)∪(⋃βBβ)=⋃(α,β)Aα∪Bβ

Proof

  • x∈(⋂αAα)∩(⋂βBβ) ⟺ ∀α∈A (x∈Aα) ∧ ∀β∈B (x∈Bβ) ⟺ ∀α∈A ∀β∈B (x∈Aα∩Bβ). Hence, (⋂αAα)∩(⋂βBβ)=⋂(α,β)Aα∩Bβ

  • x∈(⋃αAα)∪(⋃βBβ) ⟺ ∃α∈A (x∈Aα) ∨ ∃β∈B (x∈Bβ) ⟺ ∃α∈A ∃β∈B (x∈Aα∪Bβ). Hence, (⋃αAα)∪(⋃βBβ)=⋃(α,β)Aα∪Bβ

(ii) Distributivity:

  • (⋂αAα)∪(⋂βBβ)=⋂(α,β)Aα∪Bβ

  • (⋃αAα)∩(⋃βBβ)=⋃(α,β)Aα∩Bβ

Proof

  • x∈(⋂αAα)∪(⋂βBβ) ⟺ ∀α∈A (x∈Aα) ∨ ∀β∈B (x∈Bβ) ⟺ ∃α∈A ∃β∈B (x∈Aα∪Bβ). Hence, (⋂αAα)∪(⋂βBβ)=⋂(α,β)Aα∪Bβ

  • x∈(⋃αAα)∩(⋃βBβ) ⟺ ∃α∈A (x∈Aα) ∧ ∃β∈B (x∈Bβ) ⟺ ∃α∈A ∃β∈B (x∈Aα∩Bβ). Hence, (⋃αAα)∩(⋃βBβ)=⋃(α,β)Aα∩Bβ

(iii) de Morgan's laws:

  • (⋂αAα)c=⋃αAαc

  • (⋃αAα)c=⋂αAαc

Proof

  • x∈(⋂αAα)c ⟺ (x∈X)∧(x∉⋂αAα) ⟺ (x∈X)∧(∃α∈A x∉Aα) ⟺ (∃α∈A)(x∈X∧x∉Aα) ⟺ (∃α∈A)(x∈Aαc) ⟺ x∈⋃αAαc

  • x∈(⋃αAα)c ⟺ (x∈X)∧(x∉⋃αAα) ⟺ (x∈X)∧(∀α∈A x∉Aα) ⟺ (∀α∈A)(x∈X∧x∉Aα) ⟺ (∀α∈A)(x∈Aαc) ⟺ x∈⋂αAαc

 

Exercise 4

Since X and Y are non-empty, the statements u∈X and v∈Y are true for some u and v. We can then reason as follows:

(u∈X)∧(v∈Y)⇔(u,v)∈X×Y⇔⇔(u,v)∈Y×X⇔(u∈Y)∧(v∈X)

Therefore, X=Y.

 

Exercise 5

a) x∈(Ac)c ⇔ (x∉Ac) ⇔ ¬(x∈Ac) ⇔ ⇔ ¬(x∉A) ⇔ (x∈A). Hence, (Ac)c=A.

 

b) x∈(A∩Ac) ⇔ (x∈A)∧(x∉A). Reductio ad absurdum. So the original assumption is false. That is, x∉(A∩Ac). Hence, A∩Ac=∅.

 

c) x∈(A∪Ac) ⇔ (x∈A)∨(x∉A) ⇔ x∈X. The last iff statement is a consequence of the basic logical fact that True ⇔ True is valid for any two True statements. Hence, A∪Ac=X.

 

d) (Ac∪B)∩(A∩Bc) =(Ac∪B)∩(Bc∩A) =((Ac∪B)∩Bc)∩A =((Ac∩Bc)∪(B∩Bc))∩A =((Ac∩Bc)∪∅)∩A =(Bc∩Ac)∩A =Bc∩(Ac∩A) =(Bc∩∅) =∅.

Exercise 7

a) A×B≠∅ ⇔ (A≠∅)∧(B≠∅)

  • (⇒) Suppose that A×B⊆X×Y. We can now reason as follows: (a∈A)∧(b∈B) ⇔ (a,b)∈A×B ⇒ (a,b)∈X×Y ⇒ (a∈X)∧(b∈X) Hence, (A⊆X)∧(B⊆Y)

  • (⇐) Suppose that (A⊆X)∧(B⊆Y). We can now reason as follows: (a,b)∈A×B ⇔ (a∈A)∧(b∈B) ⇒ (a∈X)∧(b∈Y) ⇒ (a,b)∈X×Y Hence, A×B⊆X×Y

d) (x,y)∈(X×Y)∖(A×B) ⇔ (x,y)∈(X×Y)∧(x,y)∉(A×B) ⇔ ((x∈X)∧(y∈Y))∧(x∉A∨y∉B) ⇔ ((x∈X)∧(y∈Y)∧x∉A) ∨ ((x∈X)∧(y∈Y)∧y∉B) ⇔ ((x∈X∖A)∧(y∈Y)) ∨ ((x∈X)∧(y∈Y∖B)) ⇔ ((x,y)∈(X∖A)×Y)) ∨ ((x,y)∈X×(Y∖B)) ⇔ (x,y)∈((X∖A)×Y)∪(X×(Y∖B))

 

Exercise 8

Let {Aα ; α∈A} and {Bβ ; β∈B} be families of subsets of a set.

a) (x,y)∈(⋂αAα)×(⋂βBβ) ⟺ ∀α∈A (x∈Aα) ∧ ∀β∈B (x∈Bβ) ⟺ ∀α∈A ∀β∈B ((x,y)∈Aα×Bβ) Hence, (⋂αAα)×(⋂βBβ)=⋂(α,β)Aα×Bβ

b) (x,y)∈(⋃αAα)×(⋃βBβ) ⟺ ∃α∈A (x∈Aα) ∧ ∃β∈B (x∈Bβ) ⟺ ∃α∈A ∃β∈B ((x,y)∈Aα×Bβ) Hence, (⋃αAα)×(⋃βBβ)=⋃(α,β)Aα×Bβ
