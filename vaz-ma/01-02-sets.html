1.2 Sets and Elementary Operations on Them

 

Exercise 1

a) (A⊂C)∧(B⊂C) ⟺ ((A∪B)⊂C)

  • (⇒) Assume that LHS is true. By definition of union, x∈A∪B ⟹ (x∈A)∨(x∈B). In either case, it follows from LHS, that x∈C.

  • (⇐) Assume that RHS is true. Now, x∈A ⟹ x∈A∪B ⟹ x∈C. Where, the first implication is true by definition of union, whereas the second implication is true because we assumed that RHS is true. By a similar argument, x∈B ⟹ x∈C.

 

b) (C⊂A)∧(C⊂B) ⟺ (C⊂(A∩B))

  • (⇒) Assume that LHS is true. Then, x∈C ⟹ (x∈A)∧(x∈B). By definition of intersection, it follows that x∈A∩B.

  • (⇐) Assume that RHS is true. Now, x∈C ⟹ x∈A∩B ⟹ x∈A. Where, the first implication is true because we assumed that RHS is true, whereas the second implication is true by definition of intersection. By a similar argument, x∈C ⟹ x∈B.

 

c) CM(CMA)=A

x∈CM(CMA) ⟺ (x∉CMA) ⟺ x∈A.

 

d) (A⊂CMB) ⟺ (B⊂CMA)

(A⊂CMB) ⟺ (x∈A⇒x∈CMB) ⟺ ((x∈A)⇒(x∉B)) ⟺ ((x∈B)⇒(x∉A)) ⟺ ((x∈B)⇒(x∈CMA)) ⟺ (B⊂CMA). In the third equivalence statement, we have made use of the fact that the implication P⇒Q is equivalent to its contrapositive: ¬Q⇒¬P.

 

e) (A⊂B) ⟺ (CMA⊃CMB)

(A⊂B) ⟺ (x∈A⇒x∈B) ⟺ ((x∉B)⇒(x∉A)) ⟺ ((x∈CMB)⇒(x∈CMA)) ⟺ (CMB⊂CMA) ⟺ (CMA⊃CMB).

 

Exercise 2

(a) and (b) state respectively, that the set union and set intersection are associative. The proofs implicitly use the associativity of the logical OR and logical AND operators respectively. (c) states that the set intersection distributes over the set union. Here, the proof implicitly uses the fact that the logical AND distributes over the logical OR operator. Similarly, (d) states that the set union distributes over the set intersection and the proof implicitly uses the fact that the logical OR distributes over the logical AND operator. The proofs merely require mechanical operations with the logical operations and set operations. (a) is solved below as an example.

a) A∪(B∪C) = (A∪B)∪C := A∪B∪C

x∈A∪(B∪C) ⇔ (x∈A)∨(x∈(B∪C)) ⇔ (x∈A)∨((x∈B)∨(x∈C)) ⇔ ((x∈A)∨(x∈B))∨(x∈C) ⇔ (x∈A∪B)∨(x∈C) ⇔ x∈(A∪B)∪C.

 

Exercise 3

De Morgan's laws for set operations. Proofs are elementary as seen below.

a) CM(A∪B)=CMA∩CMB

x∈CM(A∪B) ⇔ x∉(A∪B) ⇔ (x∉A)∧(x∉B) ⇔ (x∈CMA)∧(x∈CMB) ⇔ x∈(CMA∩CMB).

Note that the second equivalence statement above follows from the fact that x∉P :⇔ ¬(x∈P), along with De Morgan's laws for logical operations proved in the previous section.

b) CM(A∩B)=CMA∪CMB

Proof is similar to (a).

Exercise 6

a) (X×Y=∅) ⇔ (X=∅)∨(Y=∅)

An obvious but useful observation: (A⇔B) ⟺ (¬A⇔¬B). Therefore, instead of proving the above equivalence, we will instead prove the following equivalence (which is equivalent to the above).

(X×Y≠∅) ⇔ (X≠∅)∧(Y≠∅).

We proceed as follows:

(X×Y≠∅) ⇔ (∃(x,y)∈(X×Y)) ⇔ ((∃x∈X)∧(∃y∈Y)) ⇔ ((X≠∅)∧(Y≠∅)).

 

For the rest of this exercise, we assume that X×Y≠∅.
