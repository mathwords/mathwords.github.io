1.1 Logical Symbolism

 

Exercise 1

Based on the truth table for A⟹B, we note in particular, that in a given context, if a statement is known to be false, then it implies the truth of any statement. For example:

(1+2=7)⟹Every circle is a square.

This construct can be used to prove that the empty set is a subset of any set. We reason as follows:

x∈∅⟹x∈A

Since the left side of the above statement is always false, it implies the truth of the statement on the right. It then follows from the definition of a subset that: ∅⊂A.

 

Exercise 2

Based on the definition of the negation (¬), we can arrive at the following conventions for handling truth tables of various operations involving negations:

  • To obtain the truth table of the negation of an operation from the truth table of the original operation, interchange the values of 1's and 0's in the table. For example, given that the first truth table below represents A∧B, then ¬(A∧B) is represented by the second table:

    A∧B(B=0)(B=1)(A=0)00(A=1)01
    ¬(A∧B)(B=0)(B=1)(A=0)11(A=1)10

  • To obtain the truth table of an operation when one of the statements involved in the operation is negated, interchange the row (or column) representing the values of the statement when true with its values when false. For example to obtain the table of A∧¬B from the table of A∧B, interchange the two columns.

    A∧B(B=0)(B=1)(A=0)00(A=1)01
    A∧¬B(B=0)(B=1)(A=0)00(A=1)10

  • To obtain the truth table of an operation when the implication is reversed, interchange the rows and columns. That is to say, move the element xij to the ji position. For example to obtain the table of B⟹A from the table of A⟹B:

    A⇒B(B=0)(B=1)(A=0)11(A=1)01
    B⇒A(B=0)(B=1)(A=0)10(A=1)11

We now use the above rules to obtain the truth tables for each of the following operations based on the truth tables given in the beginning of section 1.1.5.

a) ¬(A∧B)⟺¬A∨¬B

(B=0)(B=1)(A=0)11(A=1)10

b) ¬(A∨B)⟺¬A∧¬B

(B=0)(B=1)(A=0)10(A=1)00
