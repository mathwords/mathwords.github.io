1.1 Logical Symbolism

 

Exercise 1

  • The given truth tables serve as definitions of the negation (¬), conjunction (∧), disjunction (∨) and implication (⇒). They agree with our natural understanding of these logical operators. We make the following memorable observations:

    • A∧B is true iff both A and B are true.

    • A∨B is false iff both A and B are false.

    • A⇒B is false iff A is true and B is false.

  • A peculiar characteristic of the implication statement is worth remarking upon. Based on the truth table for A⟹B, we note in particular that if a statement A is known to be false, then the aforesaid implication statement is true for any statement B. We can occasionally use this situation to our advantage. Consider the following statement:

    x∈∅⟹x∈A

    Since the "if" part of this statement is always false, it follows that the above implication is a true statement. From the definition of "subset", it then follows that: ∅⊂A. That is to say, the empty set is a subset of any (every) set.

Exercise 2

(a) and (b) are De Morgan's laws for logical operations. (c) states that the contrapositive of the implication statement is equivalent to the implication statement. (d) can be considered the definition of the implication in terms of the 3 basic logical operations. (e) states the negation, or the contradiction of the implication statement in terms of the 3 basic logical operations.

a) ¬(A∧B)⟺¬A∨¬B

To obtain the truth table of the negation of a statement, interchange the values of 1's and 0's in the truth table of the original statement. Applying this procedure to the truth table of A∧B, we obtain the following truth table for ¬(A∧B).

(B=0)(B=1)(A=0)11(A=1)10

It is easy to see that the above truth table also represents the statement ¬A∨¬B. Indeed, this statement is false iff both A and B are true.

b) ¬(A∨B)⟺¬A∧¬B

Following the technique stated in the previous problem, we interchange the 1's and 0's in the truth table of A∨B to obtain the following truth table of ¬(A∨B).

(B=0)(B=1)(A=0)10(A=1)00

We at once observe that the above truth table also represents the statement ¬A∧¬B. Indeed, this statement is true iff both A and B are false.

c) (A⇒B)⟺(¬B⇒¬A)

Consider the following truth table of A⇒B.

(B=0)(B=1)(A=0)11(A=1)01

Observe that X⇒Y is false iff X is true and Y is false. Consequently, ¬B⇒¬A is false iff ¬B is true and ¬A is false. In other words, ¬B⇒¬A is false iff B is false and A is true. This is precisely the condition represented in the above truth table.

d) (A⇒B)⟺(¬A∨B)

X∨Y is false iff both X and Y are false. Consequently, ¬A∨B is false iff ¬A is false and B is false. In other words, ¬A∨B is false iff A is true and B is false. This corresponds precisely to the truth table of A⇒B as shown in (c) above.

e) ¬(A⇒B)⟺(A∧¬B)

Negate both sides of (d) and apply (b) to the right hand side expression.
