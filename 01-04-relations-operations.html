I.4 Relations and Operations

 

Exercise 1

We will prove that f∗:X/∼→Y/∼˙ defined by f∗([x])=[f(x)] is the function that fulfils the required conditions.

  • Indeed, (pY∘f)(x) = (pY(f(x))) = [f(x)]. Whereas, (f∗∘pX)(x) = f∗(pX(x)) = f∗([x]) = [f(x)]. Thus, f∗∘pX=pY∘f and consequently the given diagram is commutative.
  • Now suppose that g is another function that fulfils the required condition, then we require that g∘pX=pY∘f, that is, g(pX(x))=[f(x)]. But pX(x)=[x], and [f(x)]= f∗([x]). Hence, g([x])= f∗([x]). We have proved this result for all pX(x)∈X/∼. But since pX is surjective, it follows that every [x]∈X/∼ can be written as pX(x) for some x∈X. We therefore have, g=f∗ so that f∗ is unique.

 

Exercise 2

Let X be a set with at least two elements and X:=P(X)∖{X} with the inclusion order. We will prove that the identity function X→X,A↦A is not bounded. Suppose to the contrary that the given function is bounded. Then ∃B∈X,∀A∈X:A⊆B. But then we have x∈X ⇒ {x}∈X ⇒ {x}⊆B ⇒ x∈B. That is, X⊆B. Whereas, B∈P(X) ⇒ B⊆X. It follows that B=X, and hence X∈X, which is a contradiction. The original assumption is False, so that ∀B∈X,∃A∈X:A⊈B. The given function is not bounded.

 

Exercise 3

Suppose that a,b∈X:a≠b. Now consider f,g∈Funct(X,X) defined by f(x)=a and g(x)=b. We now have (f∘g)(x)=a, whereas (g∘f)(x)=b.
