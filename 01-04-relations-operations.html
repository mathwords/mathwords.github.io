I.4 Relations and Operations

 

Exercise 1

We will prove that f∗:X/∼→Y/∼˙ defined by f∗([x])=[f(x)] is the function that fulfils the required conditions.

  • Indeed, (pY∘f)(x) = (pY(f(x))) = [f(x)]. Whereas, (f∗∘pX)(x) = f∗(pX(x)) = f∗([x]) = [f(x)]. Thus, f∗∘pX=pY∘f and consequently the given diagram is commutative.
  • Furthermore the function f∗ is unique, because if g is another function that fulfils the required condition, then we require that g∘pX=pY∘f, that is, g(pX(x))=[f(x)]. But pX(x)=[x], and [f(x)]= f∗([x]). Hence, g([x])= f∗([x]). We therefore have, g=f∗ so that f∗ is unique.
